## 一句话

**你只管散步和说话，AI 替你看、替你记，回家送你一份 360° 的散步纪念品。**

---

## 我们做了什么

一款 **iOS 原生**、**音频优先** 的 AI 散步同伴 App。

用户出门：戴影石相机 + 蓝牙耳机，手机进口袋。

散步中：

- **AI 几乎不打扰**——主动开口 ≤ 3 次 / 10 分钟，且只用来"推荐附近值得看的地方"
- **被动有问必答**——你说"那是什么花？"，相机帧 + VLM 在 3 秒内给答案
- **被动捕捉**——你说"记一下"，AI 才记。**不替你判断"什么值得记"**

散步后：

- 自动生成 **30–60 秒短视频纪念品**，含 360° 视频精选 + 地图轨迹 + 对话精华字幕 + AI 海报
- 任何环节失败均**降级到长图海报**——保证至少有一份纪念品

---

## 为什么 Insta360 是核心

影石相机在产品里承担**三个不可替代的角色**：

1. **第一视角眼睛** — 给视觉模型提供"用户看到的世界"
2. **本机录制** — 高质量素材独立保存，不挤占手机带宽
3. **纪念品的灵魂** — 360° 镜头语言是这份纪念品**辨识度最高的部分**。同样一段散步，没有 360°，纪念品就是普通 vlog

**结论**：步语 = 影石相机的"日常使用入口"。每一次散步都自然产出值得回看的 360° 内容。

## 技术速览

- **架构**：ReAct Agent + 8 工具 + OpenAI 兼容 LLM 端点
- **工具**：高德 ×3、相机/视觉 ×2、交互 ×3
- **硬约束**：主动配额计数器（与 prompt 解耦的双保险）
- **降级**：相机断 / LLM 断 / 视频合成失败 — 三条独立降级路径，均已实测
- **平台**：iOS 原生（SwiftUI + AVFoundation + 高德 iOS SDK + 影石 iOS SDK）
